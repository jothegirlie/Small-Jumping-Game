# 🌙 Slime Runner Game

A simple endless runner game built with **HTML, CSS, and JavaScript** featuring a bouncing
slime character, dynamic obstacles, and sound effects.


## 🎮 Game Features

- 🟢 Play as a **bouncy slime character**
- 🔺 Avoid **spikes and gaps**
- 🌙 Night-themed animated background
- ⭐ Moving stars for atmosphere
- 🔊 Sound effects 
- 🎵 Background music
- 📱 Fully responsive canvas design
- ⚡ Increasing difficulty over time


## 🕹️ Controls

| Action | Key / Input |
|--------|------------|
| Start Game | `SPACE` or Click |
| Jump | `SPACE` or Click |
| Restart | Button on screen |
